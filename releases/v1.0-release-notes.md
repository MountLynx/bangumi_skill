## Bangumi Explorer v1.0

无认证，开箱即用的 Bangumi 查询工具。

### 功能
- 搜索番剧/书籍/游戏/音乐
- 查看条目详情（评分、标签、简介）
- 集数列表（本篇/SP/OP/ED分组）
- 当季番表（按平台分组）
- 评分排行
- 人物搜索（声优、Staff）

### 安装
1. 下载 `bangumi-explorer-v1.0.zip`
2. 解压到 OpenClaw skills 目录（如 `~/.openclaw-autoclaw/skills/bangumi_explorer/`）
3. 重启 Agent

### 使用
见 `SKILL.md` 中的命令说明。

**注意**：bangumi_explorer 仅支持公开查询，无需 Bangumi 账号。
